== Description ==

Super Rollback empowers you to revert any plugin update instantly. Works with all plugins – both free and premium.

Once installed, you will see a "Rollback" link next to each plugin on the plugins screen. Once you have updated plugins, click this link and select the version you want to rollback too. It's that easy!

Unlock all the features with [Super Rollback Pro](https://superwp.io/pricing/?ref=wporg)

= Key Features =
- Rollback any free or premium plugin to previous versions
- Lightweight with native UI
- Choose up to 5 previous versions to rollback to **[Pro]**
- Add rollback reason to keep other site admins informed **[Pro]**
- Coming Soon: Rollback even when the WP dashboard is unavailable **[Pro]**

== Installation ==

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now you will see a 'Rollback' option next to each plugin in the plugin table
4. Start updating plugins and Super Rollback will have you covered!

== Frequently Asked Questions ==

**How do I install Super Rollback?**

To install the free version of Super Rollback, follow these steps:
From your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter Super Rollback
Press install -> After installation, click Activate.

**Does Super Rollback work with all the plugins, even premium ones?**

Yes! Super Rollback is compatible with all WordPress plugins, whether free or premium. You can easily rollback any plugin update instantly.

**Do I need technical skills to use this plugin?**

Not at all! Super Rollback is designed for users of all skill levels. The interface is user-friendly, and no coding or technical expertise is required to rollback updates.

**What if I encounter issues using the plugin?**

Our dedicated support team is here to help! If you face any challenges or have questions, you can reach out to us, and we’ll assist you promptly.
